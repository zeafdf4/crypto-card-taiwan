## 用 PokeCard 之前必須知道的幾個風險

任何加密貨幣簽帳卡都有風險，PokeCard 也不例外。在決定要不要開卡之前，這幾件事你得先想清楚：

**第一，這不是真正的信用卡。** 它是簽帳卡，只能用你儲值進去的餘額，不能透支、不能分期。一旦餘額用完，卡就刷不過。

**第二，平台風險。** 你的幣儲值進 PokePay 之後，實際上是放在 PokePay 的託管帳戶裡。如果平台出問題，你的資產可能被凍結。這跟把幣放在交易所的風險是一樣的，不要把全部身家都丟進去。

**第三，幣價波動。** 如果你用 BTC、ETH 這類波動幣儲值，刷卡當下的兌換匯率會影響實際支付金額。如果不想承擔波動，建議用 USDT、USDC 這類穩定幣儲值。

**第四，手續費疊加。** 前面算過，儲值 1% + 消費 1% + 外幣 1% = 3%，這還沒算 ATM 提款的 2%。對於高頻使用者來說，這個成本會咬人。

**第五，每日限額 10 萬港幣。** 換算約 12,500 美元，對一般消費夠用，但如果你是大額出金需求（例如要刷卡買車、付房租），這個限額會卡你。

**第六，地區限制。** PokeCard 不開放美國、俄羅斯、伊拉克等地的用戶申請。台灣用戶可以正常使用，但 iOS App 在某些地區可能有商店區域限制，建議直接用網頁版或調整 App Store 區域下載。
